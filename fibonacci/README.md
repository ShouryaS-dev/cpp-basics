# Fibonacci Series

A simple C++ program that finds the Fibonacci number at a user-specified position using recursion.

## Description

This project demonstrates the use of recursive functions in C++. The program takes a position as input and calculates the corresponding Fibonacci number. It also handles special cases such as zero and negative inputs.

## Features

* Accepts user input
* Calculates Fibonacci numbers using recursion
* Handles zero and negative inputs
* Simple and beginner-friendly implementation

## Concepts Used

* Functions
* Recursion
* Conditional statements
* User input (`cin`)
* Output (`cout`)
