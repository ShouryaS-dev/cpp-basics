# Factorial Program

A simple C++ program that calculates the factorial of a user-entered number.

## Features

* Takes an integer as input
* Calculates its factorial
* Displays the result
* Uses loops and arithmetic operations

## Concepts Used

* Variables
* User input (`cin`)
* Output (`cout`)
* Loops
* Arithmetic operations

## Future Improvements

* Add input validation
* Handle very large numbers
* Implement a recursive solution
