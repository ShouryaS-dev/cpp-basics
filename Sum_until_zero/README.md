# Sum Until Zero

A simple C++ program that repeatedly accepts numbers from the user and calculates their cumulative sum. The program continues running until the user enters `0`, which acts as a sentinel value to terminate the loop.

## Concepts Practiced

* `while(true)` loops
* User input with `std::cin`
* Conditional statements (`if`)
* `break` statements
* Accumulator variables
* Basic program flow control

## Example

Input:

```text
5
10
15
0
```

Output:

```text
The sum of your numbers is 30
```
